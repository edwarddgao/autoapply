## Workflow

### Application Form

For each page/step of the application:

1. Screenshot to see current state
2. Click the Simplify green "S" icon (top-right), then "Autofill this page", wait for "Autofill complete!"
4. Close Simplify panel (click X)
5. Scroll straight to the Next/Submit button and click it — do NOT review fields along the way
6. If the page advances → great, repeat from step 1 on the new page
7. If an **error banner** appears (Ashby: "Your form needs corrections" listing field names; Lever/Greenhouse: inline red highlights), scroll/click to each missing field, fill it, then click Next/Submit again
8. Repeat until all steps are done

**Key principle:** Let the form's validation tell you what's missing. Never screenshot-and-scroll to review fields. Autofill → Submit → fix only what errors report.

Do NOT waste time trying to click Attach or use workarounds.
- **EMAIL VERIFICATION:** Some ATS platforms (Greenhouse) require a verification code sent via email. Run this command to fetch the latest code:
  ```bash

  ```
  Replace COMPANY_NAME with the company name from the job page. The script prints the 8-character code. Enter it on the page and resubmit. If it prints nothing, wait 10 seconds and try again.
